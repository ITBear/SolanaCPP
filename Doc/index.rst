Welcome to Crypto Core Webassembly's documentation!
========================================================

This project is a **WebAssembly** version of the
`Crypto core library` (_written in C++, based on `libSodium <https://github.com/jedisct1/libsodium>`_)
for the ability to work with various cryptographic functions in JavaScript projects.
